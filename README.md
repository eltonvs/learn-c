# Learn C
Aprenda C de maneira prática

## Sobre
Aprenda C de maneira didática e prática, de forma limpa e intuitiva, com apresentação de exemplos e atividades com resoluções.

## Roteiro
1. Introdução
  - Compiladores
  - Estrutura Básica
2. Saída, tipos de variáveis I e entrada.
  - Saída padrão (printf)
  - Números decimais
  - Números inteiros
  - Saída com variáveis
  - Entrada padrão (scanf)
3. Estruturas Condicionais
  - Operadores Lógicos
  - if ... else
  - else if
  - switch case
  - Lógica ternária
4. Estruturas de Repetição
  - while
  - do ... while
  - for
5. Tipos de Variáveis II
  - float
  - double
  - char
  - Conversão Explícita
  - sizeof()
6. Subrotinas
  - Funções
  - Procedimentos
7. Recursão
  - Conceitos
8. Ponteiros
  - Conceitos
9. Passagem de Parâmetros
  - Ponteiros e Funções
10. Arranjos (Arrays)
  - Vetores
  - Arranjos
  - Arranjos e Ponteiros
11. Strings
  - Leitura de Strings
  - Strings e Arranjos
  - Manipulação de Strings
12. Operadores de Bits
  - Conceitos
  - Utilidades
13. Registros (Structs)
  - Conceitos
  - Registros e Ponteiros
  - Registros e Funções
